# Beispiele

Dieses Verzeichnis enthält Beispiel-Skripte zur Verwendung der Console Table Library.

## Beispiel-Dateien

- `example.py` - Basis-Beispiele für die einfachste Verwendung
- `example_advanced.py` - Erweiterte Beispiele mit allen Features

## Ausführen

```bash
# Basis-Beispiele
python examples/example.py

# Erweiterte Beispiele
python examples/example_advanced.py
```
