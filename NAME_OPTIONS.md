# Alternative Package-Namen

Der Name `console-table` ist bereits auf PyPI vergeben. Hier sind Alternativen:

## Vorschläge:

1. **`console-table-lib`** ⭐ (empfohlen)
   - Klar und eindeutig
   - Zeigt, dass es eine Bibliothek ist

2. **`simple-console-table`**
   - Betont die Einfachheit
   - Gut für Marketing

3. **`rich-console-table`**
   - Zeigt die Rich-Integration
   - Aber könnte verwirrend sein

4. **`py-console-table`**
   - Standard Python-Präfix
   - Eindeutig

5. **`console-tables`** (Plural)
   - Einfach, aber könnte mit anderen kollidieren

6. **`table-console`**
   - Umgekehrte Reihenfolge
   - Eindeutig

7. **`ctable`** (kurz)
   - Sehr kurz
   - Aber weniger beschreibend

## Empfehlung: `console-table-lib`

Ich empfehle **`console-table-lib`**, weil:
- ✅ Eindeutig und verfügbar
- ✅ Beschreibend
- ✅ Professionell
- ✅ Zeigt, dass es eine Bibliothek ist

## Installation würde dann sein:

```bash
pip install console-table-lib
```

## Import bleibt gleich:

```python
from console_table import create  # Interner Name bleibt gleich
```
